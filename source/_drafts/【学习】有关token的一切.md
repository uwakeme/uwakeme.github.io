# 一、token概述
## （一）什么是token？
### 1、 在web领域
+ token翻译过来就是令牌的意思。
+ token是由服务器生成的一串字符串，在第一次登录成功后返回给前端，保存在浏览器或本地，在之后的请求中都会带上以供服务器校验。

### 2、 在人工智能领域
+ 在人工智能领域，尤其是自然语言处理(Natural Language Processing, NLP)中，token指的是**处理文本的最小单元或基本元素**。
